This folder contains the manually edited, production-ready version of Figure 2.

Workflow:
1. Generate raw figure: python -m evals.src.paper_figures.generate_all
2. Open raw/figure2/figure2.svg in Inkscape
3. Make manual refinements (font sizes, label positions, etc.)
4. Export to this folder: final/figure2/figure2.pdf
5. LaTeX references this folder automatically
